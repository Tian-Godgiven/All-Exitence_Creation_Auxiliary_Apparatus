## num类型属性允许的特殊字符

可以在setting中设定这些特殊的显示内容

| 值           | 含义                | 默认显示 |
| ------------ | ------------------- | -------- |
| maxN（数字） | 最大值，优先级为N   | 最大N    |
| minN         | 最小值，优先级为N   | 最小N    |
| infinityP    | 正无穷，大于任意max | +∞       |
| infinityN    | 负无穷，小于任意min | -∞       |
|              |                     |          |
