# 事物属性值特殊表达式

## 表面覆盖

所有属性值都支持表面覆盖，从而在显示时呈现不同的内容
将实际值使用`$()`包裹，放在覆盖值后方即可
注意：实际上参与各类运算例如分组表达式，时间轴系数等均使用的是实际值

~~~
覆盖值$(实际值)
~~~

~~~
年龄:永远的18岁$(99999)
~~~



## max,min最大最小值

用于数值类属性值，表示最大/最小值，其后可以紧跟数字表示最大/最小优先级，优先级越高，越大or越小，该表达式可以表面覆盖（参见上方表面覆盖），从而显示其他内容
~~~
力量:max
敏捷:min
~~~

~~~
max10 > max5
min10 < min5
~~~

~~~
力量:最强(max999)
~~~
